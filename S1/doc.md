### Сервис аутентификации

#### Регистрация

Информация требуемая для регистрации пользователя

| Параметр  | Обязательность | Тип    | Ограничение              | Значение по умолчанию |
| --------- | -------------- | ------ | ------------------------ | --------------------- |
| username  | Обязательно    | Строка | Уникальное, не пустое    | —                     |
| email     | Обязательно    | Строка | Уникальная, формат email | —                     |
| pass_hash | Обязательно    | Строка | не менее 8 символов      | —                     |

Параметры email и username должны быть уникальными 

Выходные данные

| Параметр   | Тип          |
| ---------- | ------------ |
| id         | Целое        |
| username   | Строка       |
| email      | Строка       |
| is_active  | Булево       |
| created_at | Дата и время |


#### Авторизация

Входные параметры

| Параметр  | Обязательность | Тип    | Ограничение             | Значение по умолчанию |
| --------- | -------------- | ------ | ----------------------- | --------------------- |
| username  | Обязательно    | Строка | Существует в БД         | —                     |
| pass_hash | Обязательно    | Строка | Соответствует хешу в БД | —                     |

Выходные данные

| Параметр   | Тип          | Описание              |
| ---------- | ------------ | --------------------- |
| token      | Строка       | Токен аутентификации. |
| expires_at | Дата и время | Время жизни токена    |


#### Обновление токена

Входные параметры

| Параметр | Обязательность | Тип    | Ограничение       | Значение по умолчанию |
| -------- | -------------- | ------ | ----------------- | --------------------- |
| token    | Обязательно    | Строка | Действующий токен | —                     |

Выходные данные

| Параметр   | Тип          | Описание           |
| ---------- | ------------ | ------------------ |
| token      | Строка       | Новый токен        |
| expires_at | Дата и время | Время жизни токена |



#### Сброс пароля (запрос)

Информация требуемая для создания запроса на сброс пароля

| Параметр | Обязательность | Тип    | Ограничение     | Значение по умолчанию |
| -------- | -------------- | ------ | --------------- | --------------------- |
| email    | Обязательно    | Строка | Существует в БД | —                     |

Выходные данные

| Параметр | Тип    | Описание                                      |
| -------- | ------ | --------------------------------------------- |
| message  | Строка | Текст результата для отображения пользователю |
