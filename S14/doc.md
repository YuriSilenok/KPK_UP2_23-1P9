# Вариант №14

# Сервис расчета нагрузки преподавателя (Load Calculation Service)

#### Добавить нагрузку

Информация требуемая для создания нагрузки

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|----------|----------------|-----|-------------|----------------------|
| teacher_id | Обязательно | Целое | Внешний ключ | — |
| discipline | Обязательно | Строка | Длина ≤ 200 символов | — |
| hours_per_week | Обязательно | Число с плавающей точкой | от 1 до 54 | — |
| groups_count | Обязательно | Целое | от 1 до 10 | — |
| semester | Обязательно | Целое | 1 или 2 | — |
| year | Обязательно | Целое | от 2020 до 2030 | — |
| notes | Опционально | Строка | Длина ≤ 500 символов | NULL |

Параметры teacher_id, discipline, semester, year должны быть уникальными в комбинации (один преподаватель не может вести одну дисциплину дважды в семестре)

Выходные данные

| Параметр | Тип |
|----------|-----|
| id | Целое |
| teacher_id | Целое |
| discipline | Строка |
| hours_per_week | Число с плавающей точкой |
| groups_count | Целое |
| semester | Целое |
| year | Целое |
| total_hours | Число с плавающей точкой |
| notes | Строка или NULL |

Формула расчета total_hours = hours_per_week × groups_count × 18 (недель в семестре)

#### Изменить нагрузку

Входные параметры

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|----------|----------------|-----|-------------|----------------------|
| hours_per_week | Опционально | Число с плавающей точкой | от 1 до 54 | Текущее значение |
| groups_count | Опционально | Целое | от 1 до 10 | Текущее значение |
| notes | Опционально | Строка | Длина ≤ 500 символов | Текущее значение |

> Примечание: Поля teacher_id, discipline, semester, year изменить нельзя. При изменении hours_per_week или groups_count автоматически пересчитывается total_hours.

Выходные данные

| Параметр | Тип |
|----------|-----|
| id | Целое |
| teacher_id | Целое |
| discipline | Строка |
| hours_per_week | Число с плавающей точкой |
| groups_count | Целое |
| semester | Целое |
| year | Целое |
| total_hours | Число с плавающей точкой |
| notes | Строка или NULL |

|----------|----------------|-----|-------------|----------------------|

| Параметр | Тип | Описание |
|----------|-----|----------|
| id | Целое | Уникальный идентификатор записи |

#### Получить список нагрузок

Входные параметры (все опциональны)

| Параметр | Обязательность | Тип | Описание |
|----------|----------------|-----|----------|
| teacher_id | Опционально | Целое | Фильтр по ID преподавателя |
| discipline | Опционально | Строка | Поиск по названию дисциплины |
| semester | Опционально | Целое | Фильтр по семестру (1 или 2) |
| year | Опционально | Целое | Фильтр по году |
| min_hours | Опционально | Число с плавающей точкой | Минимальное количество часов в неделю |
| max_hours | Опционально | Число с плавающей точкой | Максимальное количество часов в неделю |
| min_total | Опционально | Число с плавающей точкой | Минимальная общая нагрузка |
| max_total | Опционально | Число с плавающей точкой | Максимальная общая нагрузка |
| limit | Опционально | Целое | Максимум записей (по умолчанию 100) |
| offset | Опционально | Целое | Пагинация (по умолчанию 0) |
