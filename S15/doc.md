## Функции сервиса

1. Добавление нагрузки (закрепление преподавателя за дисциплиной в группе).
2. Получение списка назначений нагрузки по параметрам:
   a. По ID преподавателя,
   b. По ID дисциплины,
   c. По ID группы,
   d. По семестру (например, "Осень 2025"),
   e. По количеству часов (hours < n, hours = n, hours > n).
3. Получение информации о конкретном назначении нагрузки по ID.
4. Получение информации о конкретном преподавателе по ID.
5. Получение информации о конкретной дисциплине по ID.
6. Получение информации о конкретной группе по ID.
7. Изменение информации о назначении нагрузки:
   a. Изменение семестра,
   b. Изменение количества часов.
8. Удаление назначения нагрузки (по ID).
9. Получение списка всех дисциплин, закреплённых за преподавателем.
10. Получение списка всех преподавателей, ведущих определённую дисциплину.
