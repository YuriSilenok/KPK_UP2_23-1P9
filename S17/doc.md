# Room Service (Сервис аудиторий)

## Функции сервиса
1. Создание типа аудитории.
2. Создание аудитории.
3. Привязка аудитории к типу.
4. Получение информации об аудитории по ID.
5. Получение списка аудиторий по фильтрам.
6. Обновление информации об аудитории.
7. Удаление аудитории.

## ER‑диаграмма

**Сущности:**
* **Rooms (Аудитории)**
  * `id` (PK)
  * `room_number`
  * `floor`
  * `building`
  * `capacity`
* **RoomTypes (Типы аудиторий)**
  * `id` (PK)
  * `type_name`
* **RoomRoomType (Связь аудиторий и типов)**
  * `room_id` (FK → Rooms.id)
  * `room_type_id` (FK → RoomTypes.id)

## Требования к API

### 1. Добавить аудиторию (POST /rooms/)

**Информация для создания:**

| Параметр (на английском) | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
|----------------------|----------|--------------|------|-------------|--------------------|
| `room_number` | Номер аудитории | Да | string | Уникальное значение | — |
| `floor` | Этаж расположения | Да | integer | ≥ 1 | — |
| `building` | Корпус | Да | string | — | — |
| `capacity` | Вместимость (кол‑во мест) | Да | integer | > 0 | — |
| `type_ids` | Список ID типов аудитории | Нет | array of integers | Должны существовать в RoomTypes | `[]` |

**Уникальные комбинации:** `room_number + building` должны быть уникальными.

**Ответ при успешном создании:**

| Параметр (на английском) | Тип |
|----------------------|------|
| `id` | integer |
| `room_number` | string |
| `floor` | integer |
| `building` | string |
| `capacity` | integer |
| `types` | array of RoomTypeResponse |

### 2. Изменить аудиторию по ID (PUT /rooms/{room_id})

**Информация для изменения:**

| Параметр (на английском) | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
|----------------------|----------|--------------|------|-------------|--------------------|
| `room_number` | Новый номер аудитории | Нет | string | Если указан, должен быть уникальным | — |
| `floor` | Новый этаж | Нет | integer | ≥ 1 | — |
| `building` | Новый корпус | Нет | string | — | — |
| `capacity` | Новая вместимость | Нет | integer | > 0 | — |
| `type_ids` | Новые ID типов | Нет | array of integers | Должны существовать в RoomTypes | — |

**Ответ при успешном изменении:** аналогичен ответу при создании.

### 3. Удаление аудитории по ID (DELETE /rooms/{room_id})

**Ответ:** `boolean` (`True` — удалено, `False` — не найдено).


### 4. Получить аудиторию по ID (GET /rooms/{room_id})

**Ответ при успешном поиске:**

| Параметр (на английском) | Пояснение | Тип |
|----------------------|----------|------|
| `id` | Уникальный идентификатор | integer |
| `room_number` | Номер аудитории | string |
| `floor` | Этаж | integer |
| `building` | Корпус | string |
| `capacity` | Вместимость | integer |
| `types` | Типы аудитории | array of objects |

### 5. Получить список аудиторий (GET /rooms/)

**Параметры фильтрации:**

| Параметр (на английском) | Пояснение | Тип | Описание |
|----------------------|----------|------|----------|
| `building` | Фильтр по корпусу | string | Возвращает аудитории только указанного корпуса |
| `floor` | Фильтр по этажу | integer | Возвращает аудитории только указанного этажа |
| `min_capacity` | Минимальная вместимость | integer | Возвращает аудитории с вместимостью ≥ указанному значению |
| `type_id` | Фильтр по типу аудитории | integer | Возвращает аудитории указанного типа |

**Ответ в виде списка аудиторий:**

| Параметр (на английском) | Тип |
|----------------------|------|
| `id` | integer |
| `room_number` | string |
| `floor` | integer |
| `building` | string |
| `capacity` | integer |
| `types` | array of RoomTypeResponse |
